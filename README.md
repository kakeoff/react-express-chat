# SocketIO Chat

> Simple chat app with rooms made with ReactJS, Express, Socket.io

## Preview

![image](client/src/assets/images/preview.png)
![image](client/src/assets/images/preview-2.png)

## Features

- ReactJS
- React-router-dom
- ESLint

- ExpressJS
- Socket.io
- ESLint

## Client setup

Install dependencies:

```
npm install
```

npm run start

```

## Server setup

Install dependencies:

```

npm install

```
npm run dev
```
